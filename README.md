# Git for Data Science

This repository contains scripts for our MLOps course on MLFlow part 1.

- `demo_app.py`: It contains all the code which was used for demonstrating bits and pieces of mlflow until adding tags. 
- `demo_client.py`: It contains the code for "Finding the Best Model Programmatically" section
- `best_model.py`: It contains the code for "Comparing runs" section
- `project.py`: It has the full script for hands on project section
- `project_client.py`: It has the full script for hands on project client section
